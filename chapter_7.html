Setup

Let us use sage to define position and momentum operator. 

R.<x> = QQbar[]
W = DifferentialWeylAlgebra(R)
(x, dx) = W.gens()
p = -i*dx

Now let us define the commutator operation, the annihilation (\(a\)), and creation operator (\(a^\dagger\)). 

def com(a, b):
    """Commutator [a,b] of a and b."""
    return a*b - b*a

a = (x + i*p) / sqrt(2)
ad = (x - i*p) / sqrt(2)

assert com(x, p) == i
assert com(a, ad) == 1
